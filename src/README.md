# 🎈 Argaman - Воздушные шары и фотосъемка

Сайт для праздничных мероприятий с фокусом на брит-милах и бар-мицву.

## 🎨 Дизайн-система Argaman

**Винные темные цвета:**
- `#6b1a3d` - Глубокий винный (основной)
- `#4d184d` - Темный пурпур (вторичный)
- `#8b2e3f` - Винно-красный (третичный)
- `#c9a9b8` - Приглушенный лиловый (акцент)

## 🚀 Быстрый старт

### Локальная разработка

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр production сборки
npm run preview
```

## 📦 Деплой на Hostinger

### Автоматический деплой (Git Deployment)

1. Push в GitHub автоматически деплоит на Hostinger
2. Скрипт `deploy.sh` выполняется автоматически
3. Сайт обновляется в течение 2-3 минут

### Ручной деплой

```bash
npm run build
# Загрузите содержимое папки dist/ в public_html через FTP
```
